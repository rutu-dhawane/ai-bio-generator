# ai-bio-generator

AI Bio Generator is a Streamlit-based web application that generates professional and personalized biographies for different platforms such as LinkedIn, Twitter, Instagram, and Personal Websites.

## Features

- Generate platform-specific bios
- Professional and personalized content
- Simple and user-friendly interface
- Supports LinkedIn, Twitter, Instagram, and Personal Website formats

## Technologies Used

- Python
- Streamlit

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
